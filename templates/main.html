Input an IP address to validate:


IP address must be in the following format in order to be valid:
  • Be numerical and between 4 and 12 digits long, with four parts separated by '.'
  • First block between 10 and 255
  • Second, third and fourth block between 0 and 255

This application will also check if the IP address is a known Tor exit node, and it's approximate location if avaliable.


This is a web-based program to check the formatting of a standard IPv4 IP address and whether it is a known Tor exit node.
